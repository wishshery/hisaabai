# HisaabAI — حساب AI

AI-powered Urdu bookkeeping for Pakistani small traders. Voice-first, powered by Claude Cowork (Anthropic API).

## Features
- Voice entry in Urdu/Punjabi parsed by Claude AI
- - Daily earnings, expenses and profit dashboard
  - - Weekly reports with AI insights
    - - Microloan credit score profile
     
      - ## Stack
      - React + Vite · Claude Sonnet 4 · GitHub Pages
     
      - Live: https://wishshery.github.io/hisaabai/
